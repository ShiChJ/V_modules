# V_modules

Data-Processing Modules for Data Sources

## Completed Modules

- Module1(v1.01):
	- First Edition: 2017/5/3
	- Functions:
		- Calculating coverage ratio for each field to evaluate its availablity and visualization with a barplot;
		- Calculating metrics including precision and k-s statistics.

- Module2(v1.0):
	- First Edition: 2017/5/4
	- Functions:
		- Calculating statistics for specific field, incluing hit ratio(bad customer), accidentally injure ratio(good customer) and k-s statistics;
		- Drawing PDF/CDF curve for specific field.
